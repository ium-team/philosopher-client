# Philosopher Client

Next.js 16 기반 프론트엔드 초기 베이스 프로젝트다.

## Commands

```bash
pnpm dev
pnpm lint
pnpm typecheck
pnpm build
pnpm format
```

## Environment

`.env.example` 를 기준으로 `.env.local` 을 구성하면 된다.
